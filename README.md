# Note Taker

## Purpose
A simple app to take notes.

## How to Use
* Enter a note title and description
* Click the "Save" icon on the top right of the page
* Select a note from the left to view the description
* Click + to add a new note

## Built With
* HTML
* CSS
* Bootstrap
* fontawesome
* Express
* Node
